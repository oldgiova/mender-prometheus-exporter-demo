# mender-prometheus-exporter-demo
Demo of a Prometheus Exporter for Mender

## Expose the service
```
kubectl expose pod/roberto-prometheus-poc --type="ClusterIP" --port 8000 --name metrics
```

## Enable Scraping

```
cat <<EOF | kubectl apply -f -
apiVersion: monitoring.coreos.com/v1
kind: ServiceMonitor
metadata:
  labels:
    run: roberto-prometheus-poc
  name: roberto-prometheus-poc
  namespace: staging
spec:
  endpoints:
  - interval: 30s
    path: /
    port: metrics
    scheme: http
  selector:
    matchLabels:
      run: roberto-prometheus-poc
EOF
```
